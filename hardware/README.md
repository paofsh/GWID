# Hardware

This directory contains the open hardware design for the
GWID (General-purpose WiFi-connected Indicator Device).

The hardware is licensed under the **CERN Open Hardware Licence v2 - Permissive
(CERN-OHL-P)**. See [`LICENSE-HARDWARE`](/LICENSE-HARDWARE) in the project root for details.

See [`GWID-Hardware-Build.md`](GWID-Hardware-Build.md) for schematics.
